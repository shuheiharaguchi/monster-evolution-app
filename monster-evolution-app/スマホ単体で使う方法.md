# スマホ単体で使う方法

このアプリはPWA対応済みです。

## いちばん安定する方法

`outputs/monster-evolution-app` フォルダをHTTPSで公開できる場所に置きます。

例:
- GitHub Pages
- Netlify
- Vercel
- 自宅NASやレンタルサーバーのHTTPS公開フォルダ

スマホで公開URLを開いたあと、ホーム画面に追加します。

## iPhone

1. SafariでURLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. 追加したアイコンから開く

## Android

1. ChromeでURLを開く
2. メニューを開く
3. 「ホーム画面に追加」または「アプリをインストール」を押す

## 注意

Macの `localhost` や `192.168.x.x` で開く方法は、Macが起動している間だけ使えます。
スマホ単体で使うには、HTTPSの公開URLから一度開いてホーム画面に追加してください。
